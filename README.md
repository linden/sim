# Sim
a simple client and daemon for simnet.

# Usage
## `simd`
Running `simd` is pretty straightforward. Since it's blocking I'd recommend running it in a subshell.
```sh
simd &
```

## `sim`
### Send Sats
Send `100` sats to `x`.
```sh
sim send x 100
```

### Mine Blocks
Mine 100 blocks
```sh
sim mine 100
```
